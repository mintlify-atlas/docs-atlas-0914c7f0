## Model Structure

```typescript
const Todo = S.Struct({
  id: S.String,
  text: S.String,
  completed: S.Boolean,
  createdAt: S.Number,
})

const Filter = S.Literal('All', 'Active', 'Completed')

const NotEditing = ts('NotEditing')
const Editing = ts('Editing', {
  id: S.String,
  text: S.String,
})

const Model = S.Struct({
  todos: S.Array(Todo),
  newTodoText: S.String,
  filter: Filter,
  editing: S.Union(NotEditing, Editing),
})
```

The model tracks:
- The list of todos
- Current input text for new todos
- Active filter
- Editing state (either not editing, or editing a specific todo)

## Local Storage Persistence

### Saving Todos

Whenever todos change, they're automatically saved:

```typescript
const saveTodos = (todos: Todos): Command<typeof SavedTodos> =>
  Effect.gen(function* () {
    const store = yield* KeyValueStore.KeyValueStore
    yield* store.set(TODOS_STORAGE_KEY, S.encodeSync(S.parseJson(Todos))(todos))
    return SavedTodos({ todos })
  }).pipe(
    Effect.catchAll(() => Effect.succeed(SavedTodos({ todos }))),
    Effect.provide(BrowserKeyValueStore.layerLocalStorage),
  )
```

### Loading Todos

On startup, todos are loaded from local storage:

```typescript
const flags: Effect.Effect<Flags> = Effect.gen(function* () {
  const store = yield* KeyValueStore.KeyValueStore
  const maybeTodosJson = yield* store.get(TODOS_STORAGE_KEY)
  const todosJson = yield* maybeTodosJson

  const decodeTodos = S.decode(S.parseJson(Todos))
  const todos = yield* decodeTodos(todosJson)

  return { todos: Option.some(todos) }
}).pipe(
  Effect.catchAll(() => Effect.succeed({ todos: Option.none() })),
  Effect.provide(BrowserKeyValueStore.layerLocalStorage),
)
```

The flags are passed to `init` to initialize with persisted data:

```typescript
const init: Runtime.ElementInit<Model, Message, Flags> = flags => [
  {
    todos: Option.getOrElse(flags.todos, () => []),
    newTodoText: '',
    filter: 'All',
    editing: NotEditing(),
  },
  [],
]
```

## CRUD Operations

### Creating Todos

```typescript
AddedTodo: () => {
  if (String.isEmpty(String.trim(model.newTodoText))) {
    return [model, []]
  }
  return [model, [generateTodoData(String.trim(model.newTodoText))]]
}

GotNewTodoData: ({ id, timestamp, text }) => {
  const newTodo: Todo = {
    id,
    text,
    completed: false,
    createdAt: timestamp,
  }
  const updatedTodos = [...model.todos, newTodo]
  return [
    evo(model, {
      todos: () => updatedTodos,
      newTodoText: () => '',
    }),
    [saveTodos(updatedTodos)],
  ]
}
```

### Updating Todos

```typescript
ToggledTodo: ({ id }) => {
  const updatedTodos = Array.map(model.todos, todo =>
    todo.id === id
      ? evo(todo, { completed: completed => !completed })
      : todo,
  )
  return [
    evo(model, {
      todos: () => updatedTodos,
    }),
    [saveTodos(updatedTodos)],
  ]
}
```

### Deleting Todos

```typescript
DeletedTodo: ({ id }) => {
  const updatedTodos = Array.filter(model.todos, todo => todo.id !== id)
  return [
    evo(model, {
      todos: () => updatedTodos,
    }),
    [saveTodos(updatedTodos)],
  ]
}
```

## Inline Editing

The editing state is a discriminated union:

```typescript
StartedEditing: ({ id }) => {
  const todo = Array.findFirst(model.todos, t => t.id === id)
  return [
    evo(model, {
      editing: () =>
        Editing({
          id,
          text: Option.match(todo, {
            onNone: () => '',
            onSome: t => t.text,
          }),
        }),
    }),
    [],
  ]
}

SavedEdit: () =>
  M.value(model.editing).pipe(
    M.tagsExhaustive({
      NotEditing: () => [model, []],
      Editing: ({ id, text }) => {
        if (String.isEmpty(String.trim(text))) {
          return [
            evo(model, {
              editing: () => NotEditing(),
            }),
            [],
          ]
        }
        const updatedTodos = Array.map(model.todos, todo =>
          todo.id === id
            ? evo(todo, { text: () => String.trim(text) })
            : todo,
        )
        return [
          evo(model, {
            todos: () => updatedTodos,
            editing: () => NotEditing(),
          }),
          [saveTodos(updatedTodos)],
        ]
      },
    }),
  )
```

The view shows different UI based on editing state:

```typescript
const todoItemView =
  (model: Model) =>
  (todo: Todo): Html =>
    M.value(model.editing).pipe(
      M.tagsExhaustive({
        NotEditing: () => nonEditingTodoView(todo),
        Editing: ({ id, text }) =>
          id === todo.id
            ? editingTodoView(todo, text)
            : nonEditingTodoView(todo),
      }),
    )
```

## Filtering

```typescript
const filterTodos = (todos: Todos, filter: Filter): Todos =>
  M.value(filter).pipe(
    M.when('All', () => todos),
    M.when('Active', () => Array.filter(todos, todo => !todo.completed)),
    M.when('Completed', () => Array.filter(todos, todo => todo.completed)),
    M.exhaustive,
  )
```
